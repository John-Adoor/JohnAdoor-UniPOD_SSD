2D Design: X-Shaped Laptop Stand

Creating Technical Drawings in CorelDraw for Single-Material X-Stand

2D Design Process Summary

  • Design: Simple X-shaped stand with two crossed parts
  • Material Options: Wood OR Acrylic (single material product)
  • Components: Only 2 identical parts + rubber pads
  • Key Features: Adjustable height, collapsible, tool-free assembly
  • Next Phase: 3D modeling in SolidWorks
 

Introduction

This week focuses on developing the selected X-shaped laptop stand concept from Week 3 into detailed 2D technical drawings using CorelDraw. The design maintains the simplicity of the original concept - two identical parts that cross to form a stable X-shaped stand.

The product will be manufactured from a single material - either wood or acrylic - creating a minimalist, elegant solution that requires no additional hardware for assembly.

Material Options

Single Material Production Options

The X-shaped stand will be produced using either wood or acrylic as the primary material, not a combination.

Wood Option
  • Material: Solid hardwood (12mm thickness)
  • Finish: Natural wood with protective coating
  • Weight: ~600g
  • Advantages: Natural aesthetic, durability, warmth
  • Manufacturing: CNC routing or laser cutting
Acrylic Option
  • Material: Clear cast acrylic (8mm thickness)
  • Finish: Polished edges, optional tints
  • Weight: ~400g
  • Advantages: Modern look, lightweight, transparent
  • Manufacturing: Laser cutting with precision
Design Features Comparison
Common Features (Both Materials)
  • Two identical crossed parts
  • Multiple height adjustment positions
  • Interlocking notch system
  • Collapsible for storage
  • Rubber pads for stability
  • No tools required for assembly
  • Compact folded size: 300mm × 30mm × 50mm
Material-Specific Considerations
  • Wood: Requires slightly thicker profile (12mm) for strength
  • Acrylic: Can use thinner profile (8mm) due to material strength
  • Wood: Natural grain variations add character
  • Acrylic: Consistent appearance, modern aesthetic
  • Both: Same functional design, different visual appeal

CorelDraw Design Process

Creating the Two-Part X-Shaped Stand

Systematic approach to designing the simple yet functional X-shaped stand.

Step 1: Basic Part Geometry

Create the fundamental shape of the two identical parts:

  • Draw main body: 280mm length × 60mm maximum width
  • Create tapered design narrowing to 30mm at ends
  • Add center pivot slot for crossing mechanism
  • Design interlocking notches at 3 positions for height adjustment
  • Apply consistent radius (5mm) to all corners
Single part design showing tapered profile and interlocking features
Step 2: Interlocking Mechanism

Develop the crossing and locking system:

  • Create center crossing slot (60mm × 8mm)
  • Design three height adjustment notches along length
  • Add rubber pad placement areas (4 positions)
  • Ensure smooth engagement/disengagement of parts
  • Verify structural stability in all positions
Step 3: Assembly Configuration

Document how the two parts interact:

  • Show parts crossing at 90-degree angle
  • Illustrate all height adjustment positions
  • Demonstrate collapsed storage configuration
  • Show rubber pad installation locations
  • Indicate laptop placement and weight distribution
Two parts crossed to form stable X-shaped stand
Step 4: Technical Documentation

Create comprehensive manufacturing drawings:

  • Single part drawing with all dimensions
  • Assembly drawing showing crossed configuration
  • Height adjustment positions documentation
  • Material specifications and thickness
  • Rubber pad installation guide
Step 5: Manufacturing Considerations

Optimize design for production:

  • Ensure design works for both wood and acrylic
  • Specify appropriate corner radii for material
  • Include tolerance specifications
  • Add grain direction for wood version
  • Specify edge finishing requirements
Complete technical drawing ready for manufacturing

Reflection

Developing the 2D technical drawings for this simplified X-shaped laptop stand has reinforced the value of minimalist design. By reducing the product to just two identical parts, we've created a solution that is not only elegant but also highly manufacturable and user-friendly.

The single-material approach (either wood or acrylic) simplifies production while maintaining the core functionality of the original concept. Both material options offer distinct advantages: wood provides warmth and natural appeal, while acrylic offers a modern, lightweight alternative. The design accommodates both materials with only minor adjustments to thickness and manufacturing specifications.

Creating the interlocking mechanism in CorelDraw required careful attention to tolerances and engagement surfaces. The notch system for height adjustment needed to provide secure locking while allowing easy adjustment by the user. The tapered design of the parts ensures visual appeal while maintaining structural integrity.

The collapsible feature was particularly challenging to implement in a two-part design. Ensuring that the parts would nest compactly while maintaining all functional features required several iterations. The final design achieves a folded size that is highly portable while being instantly ready for use.

These comprehensive 2D drawings provide a solid foundation for prototyping and eventual production. For the next phase, I will transition to SolidWorks for 3D modeling, where I can create virtual prototypes, perform stress analysis, and optimize the design for manufacturing. The parametric capabilities of SolidWorks will allow for easy iteration between the wood and acrylic versions while maintaining the same core geometry.

This simplified X-shaped stand demonstrates how effective design doesn't require complexity. With just two parts and no additional hardware, we've created a functional, adjustable laptop stand that meets all the user needs identified in our design thinking process.
